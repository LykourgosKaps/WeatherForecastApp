# 🌍 Climate Hazard Map

Interactive React application that visualizes climate hazards across different regions.

## 🚀 Features
- 🗺️ Interactive map (Leaflet + Mapbox)
- 🎯 Multi-filter system:
  - Hazard (Rain, Snow, Drought)
  - Probability (Low, Medium, High)
  - Magnitude (Low, Medium, High)
- 📍 Dynamic markers based on selected filters
- 🔄 Reset filters after each apply

## 🛠️ Tech Stack
- React (Vite)
- Leaflet
- Mapbox
- JavaScript (ES6)
